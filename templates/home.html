Web App

The Comp Finder web-app allows you to find all the WCA cubing competitions in states nearby to you. Use it to find competitions near you right now by clicking the "Try It" tab.

Github Page


Command Line Tool

Comp Finder is also available as a command-line tool. This runs a daemon process, which means that it is always running in the background along with all your other computer tasks. It updates a file containing information about the competitions every 30 minutes.
